# Features and Roadmap

## Current Features

### Khatas

- Amount + note quick input format
- Inline editing and delete via swipe gestures
- Haptic feedback and toast status

### Calendar

- Monthly grid layout
- Tap day to inspect entries
- Long-press and drag to select date ranges
- Scrollable selected-range entry list

### Insights

- This week total
- Last week total
- 7-day trend percentage
- Current month spend
- Top spend items

### Settings

- Currency switching
- Export data (JSON) to custom directory/file
- Import data (JSON) from custom location
- Clear all local data

### UX / Theming

- Minimal dark UI
- Inter variable + monospace typography mapping
- Configurable haptics and toast tokens

## Upcoming Features

- Category-specific insights
- Advanced filters (date, note, amount)
- Undo delete and safer destructive actions
- Recurring entry templates
- App lock / biometric enhancements
- CSV export option
- Better import validation and conflict preview
